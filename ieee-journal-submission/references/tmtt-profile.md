# T-MTT 投稿画像

目标期刊：`IEEE Transactions on Microwave Theory and Techniques`。

官方入口：<https://mtt.org/publications/t-mtt/information-for-authors/>。本画像最后核对：`2026-07-14`。

本文件只保存已确认的阶段差异和本次实战暴露的冲突。每次新投稿、返修和录用后提交前重新访问官方页面。

## 初投稿

当前公开指南写明：

- 使用 IEEE Transactions 模板；
- 初投稿为单个 PDF；
- PDF 低于 1 MB；
- 不接受 ZIP。

本次真实 Research Exchange 页面同时要求 `main document LaTeX source`，与公开指南存在冲突。处理规则：

1. 记录公开指南原文和访问日期；
2. 记录当前页面字段和文件类型；
3. 当前页面控制当前操作；
4. 不把本次 source 要求推广到下一篇 T-MTT 稿件；
5. 必要时向编辑部询问，不猜测。

## 同行评审与返修

- 当前指南说明 T-MTT 采用 single-anonymous peer review。
- 返修材料包括无高亮修订稿、高亮修订稿和逐条 `Reply to Reviewers`。
- 具体截止日期、可延期规则和文件命名以决定信与返修页面为准。

## 录用后最终文件

当前指南把最终生产文件与初投稿明确分开。录用后要求包括：

- LaTeX 或 Word 源文件；
- 嵌入全部图表的完整 PDF；
- 每幅图、表和照片的独立文件；
- Page Charge Authorization；
- 最终文件上传后的电子 copyright 流程。

不要把初投稿 source 包直接复用为最终生产包。转 `latex-paper` 按当次决定信和生产页面重新打包。

## Graphical Abstract

当前指南为 `strongly encouraged`，不是已确认的强制项。建议格式为 PDF/JPG/PNG、横向、约 1:1 至 2:1、300 ppi。实际是否显示为必填，以当前投稿页面为准。

## ORCID、数据和代码

- IEEE 通用作者工具页说明期刊作者需要 ORCID；投稿和校样都会使用。
- IEEE 与 T-MTT 鼓励共享数据和代码；截至核对日，未找到 T-MTT 强制公开数据或代码的规则。
- `Data` / `Code` 页面字段的精确定义和 Yes/No 后续材料要求属于页面动态配置。先读帮助文字，再根据实际公开共享状态选择。

## 保持 Pending

- 是否出现 recommended reviewers 字段及数量；
- opposed reviewers 是否可选、是否需要理由；
- Data/Code 问题的当前精确定义；
- Research Exchange 是否为当前稿件启用转投；
- 当前页面是否仍要求初投稿 LaTeX source。
