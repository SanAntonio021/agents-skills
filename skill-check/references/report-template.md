# Report Template

周报 `weekly/<date>.md` 固定按这个顺序输出：

1. `扫描范围`
2. `当前活跃 skill`
3. `摘要计数`
4. `阻塞项`
5. `真重复候选`
6. `名字漂移`
7. `边界交叉候选`
8. `路径漂移`
9. `空壳/破损项`
10. `建议动作`
11. `无需动作说明`

`summary.json` 与周报保持同一批结论，但结构化保留：

- 扫描根目录与报告目录
- 目录模型（flat / segmented）
- 活跃 skill 清单
- 各检查项计数
- 各类 findings 详情
- 建议动作与无需动作说明

如果某个检查项没有发现问题，也要在周报里显式写 `- 无`，不要省略该章节。
