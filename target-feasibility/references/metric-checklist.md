# 指标论证检查清单

使用本清单检查每一条技术指标。

## 1. 指标对象

- 这条指标属于哪一类指标？
- 它对应的对象到底是什么？
- 它是单路、单节点、聚合链路还是整系统？
- 它是峰值、持续值、典型值还是保证值？
- 它是实验指标还是工程部署指标？

## 2. 指标边界

- 是否包含网络层及以上处理？
- 是否包含编解码？
- 是否包含路由、转发、存储或调度？
- 是否包含建链、接入或握手过程？

## 3. 定量依据

- 是否存在真实的分解路径？
- 是否列出了主要贡献项？
- 数值是实测、估算，还是来源引用？
- 是否保留了足够的工程余量？

## 4. 工程真实性

- 现有器件或公开结果能否直接支撑？
- 是否只在特殊条件下成立？
- 该指标是否容易被评审认为是反向凑出来的？

## 5. 表述质量

- 指标本身是否已经把对象说清楚？
- 表述中是否仍有隐含假设？
- 是否尽量保留了用户原有结构？

## 6. 来源质量

- 每个外部数值是否都能找到引用？
- 来源是否为一手来源？
- 来源是否真正对应当前指标，而不是相近指标？
