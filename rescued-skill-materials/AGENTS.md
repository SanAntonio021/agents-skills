# rescued-skill-materials 规则

## 作用

这里专门存放从已删除 skill 中救回来的参考材料和脚本。
这里不是新的 skill 目录，不放 `SKILL.md`，也不作为技能入口。

## 目录结构

- `references/`：保留值得复用的文段、模板、说明文档
- `scripts/`：保留值得复用的工具脚本

## 命名规则

1. 保留原 skill 名作为一级目录，方便追溯来源。
2. 目录内尽量保留原文件名，避免二次改名造成来源混乱。

## 清理原则

1. 这里只保留确认有复用价值的材料，不把整个已删 skill 原样搬回来。
2. 如果某份材料后来被正式并入现有 skill，可再从这里删掉对应副本。
3. 删除这里的文件前，先确认内容已经迁到更合适的位置。
