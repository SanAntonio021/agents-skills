# 报告与台账补录清单

## 测试报告最小内容

- 测试目的
- 链路文字描述
- 仪器与通道
- `AWG OFF` 底噪
- `AWG ON` 带内信号功率
- 开关功率比
- `MER`
- `BER`
- `SyncPSNR`
- 结果目录和图像链接

## 台账核对顺序

1. 仪器台账
2. 线缆台账
3. 接头台账
4. 射频模块台账
5. 常用 DUT 配置

## 补录规则

- 精确条目优先，最近似条目次之
- 若用了“最近似”，报告里要明确说明
- 测试链路可补入：
  - 常用 DUT 配置
  - 测试汇总文档
  - 台账备注

## 中文命名建议

- 报告文件名优先中文
- 标题优先中文
- 需要保留英文的仅限：
  - 仪器型号
  - 序列号
  - CSV 文件名
  - 图片文件名
