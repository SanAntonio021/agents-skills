# 终稿精修规则

当用户已经有一版可交付初稿，当前步骤明确是提交前精修、导出前整理或审稿前整理时，按此文件执行。

## 典型输入

- 论文草稿文件（Markdown、Word 或 LaTeX）
- 可选上下文：大纲文件、证据草稿、参考文献库

## 精修规则

1. 保持原有 citation keys。
2. 引用保留在原来的小节内。
3. 弱证据对应克制结论。
4. 去掉模板化套话、过渡废话和重复句式。
5. 先确保引用和数据没被改错，再做术语统一和冗余压缩。
6. 遇到实验结果类段落，先收紧该段真正要证明的对象，再改句子。
7. 结果段优先回答"实验想证明什么、图里看到了什么、这说明了什么"；方法段只保留系统连接、关键参数、测量链路和分析指标。
8. 频率、功率、工作点、输入范围、步进和代表性谱线等参数，要和图中可观测量一一对应。
9. 代表性指标必须交代选择依据，例如"幅度最高""最稳定""最能代表其余分量"。
10. 统一术语时按层级分工；过程、信号、谱线和指标各自用词应固定。
11. 去掉预防性辩护和自我解释式句子，改成客观陈述句。
12. 图注写成正文结果的压缩版，说明标签对应的读法。
13. 中英双语场景下，英文稿按目标期刊风格二次重写。

如果用户明确要求对齐正式技术文体或英文论文风格，再按需读取 [writing-samples.md](writing-samples.md)；只借句式密度、术语分层和信息推进方式，具体内容仍以当前稿件为准。

## 实验段落精修

1. 先找主线：这段到底是在证明性能、保护能力、机理，还是仅在交代配置；如果主线偏了，先收紧标题和首句。
2. 确保信号链描述完整连贯：把源信号、干扰信号、变频关系、图中谱线和最终分析指标写成一条读者能顺着走通的链路。
3. 再切结果和方法：结果段保留设计动机、图中现象和结论；方法段只保留连接关系、参数设置、测量链路和选取的分析指标。
4. 最后做术语统一：先统一高频术语，再统一动词和图注风格；如果还有英文版本，按投稿风格单独再修一轮。
